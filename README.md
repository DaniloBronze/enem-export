# enem-export

Assets e JSON exportados das provas do ENEM.

## Estrutura

- `assets/enem/{ano}/{cor}/dia{n}/` — imagens das questões
- `data/enem-{ano}-{cor}-dia{n}.json` — questões com URLs públicas
